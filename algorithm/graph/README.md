## 图
### 概念
一个图 G = (V, E) 由以下元素组成
- V: 一组定点
- E: 一组边

1. 相邻顶点: 一条边连接起来的顶点叫做相邻顶点
2. 度: 一个顶点的度是其相邻顶点的数量
3. 有向图: 边是有方向的叫做有向图,两个顶点双向上都存在路径则该图是强联通的
4. 无向图: 边上是没有方向的
5. 权: 就是边是否赋予值,赋值了叫有权,不赋值叫无权
6. 邻接表/邻接矩阵: 都是通过展示两个点表示两点是否联通来表示图
7. 关联矩阵: 通过展示点在哪条边上来表示图
