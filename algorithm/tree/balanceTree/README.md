## 平衡搜索二叉树
出现平衡搜索二叉树的原因是如果二插搜索树的节点是1,2,3,4,5那么树就退化成了链表
![搜索树退化成链表](./pic/1.webp)

这个时候我们就要将搜索二叉树平衡一下成为平衡搜索二叉树

### 平衡因子
如何来判断这个节点是否平衡这个时候就需要引入一个平衡因子的概念

平衡因子 = 左子树深度 - 右子树深度

当所有节点平衡因子为 0, 1, -1 时说明这颗二叉搜索树是平衡二插搜索树
